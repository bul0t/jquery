# Атрибуты
Работаем с атрибутами в CSS.

Получаем значение атрибута:

    $('a').attr('href')

Устанавливаем значение атрибута:
    
    $('a').attr('href', 'https://yandex.ru')

Устанавливаем сразу несколько значений:

    $('img').attr({

    })
